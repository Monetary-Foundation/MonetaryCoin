ComplianceStore

contract ComplianceStore

is MultihashStore

Enables to store optional compliance data supports one multihash per address more information about multihash can be found on https://github.com/multiformats/multihash Important hash functions: (Name, hashFunction, size in bytes) (sha1, 0x11, 20) (sha2-256, 0x12, 32) (sha3-256, 0x16, 32) Full list can be found on https://github.com/multiformats/multihash/blob/master/hashtable.csv.

Source: storage/ComplianceStore.sol
