GDPOraclizedToken

contract GDPOraclizedToken

is MineableToken

An interface for the GDP Oracle to control the mining rate. For security reasons, two distinct functions were created: setPositiveGrowth() and setNegativeGrowth().

Source: token/ERC20/GDPOraclizedToken.sol
